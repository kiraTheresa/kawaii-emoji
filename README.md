# 表情包显示网站

一个基于React和Express的表情包显示网站，支持按文件夹浏览、图片预览和搜索功能，界面风格为可爱的kawaii动漫风格。

## 功能特性

- 📁 **按文件夹分类**：自动根据文件夹名称对表情包进行分类
- 🔍 **搜索功能**：支持按文件名搜索表情包
- 🖼️ **图片预览**：点击图片可查看大图
- 📱 **响应式设计**：适配不同屏幕尺寸，从手机到2560px超大屏幕
- 🌈 **kawaii动漫风格**：使用明亮柔和的配色和可爱的设计元素
- 🚀 **一键启动**：方便快捷的启动脚本
- 支持多种图片格式：jpg, png, gif, webp, svg, bmp

## 技术栈

- **前端**：React + Vite
- **后端**：Express
- **样式**：CSS3 (kawaii动漫风格)
- **工具**：Node.js, npm

## 快速开始

### 环境要求

- Node.js >= 20.18.0 (推荐20.19.0+)
- npm >= 10.0.0

### 安装依赖

```bash
npm install
```

### 配置环境变量

1. 在项目根目录找到 `.env` 文件
2. 修改 `EMOJI_DIR` 为你的表情包文件夹路径，例如：

```env
EMOJI_DIR = D:\A_收藏集\表情包
```

### 启动应用

#### 方法一：一键启动（推荐）

```bash
npm run start
```

这个命令会同时启动后端服务器和前端开发服务器。

#### 方法二：分别启动

1. 启动后端服务器：
   ```bash
   npm run server
   ```

2. 在另一个终端启动前端开发服务器：
   ```bash
   npm run dev
   ```

### 访问应用

在浏览器中输入以下地址：
```
http://localhost:5173
```

## 使用说明

1. **浏览表情包**：在左侧文件夹列表中点击文件夹名称，右侧会显示该文件夹下的所有表情包
2. **搜索表情包**：在顶部搜索框中输入关键词，会实时过滤当前文件夹下的表情包
3. **查看大图**：点击任意表情包图片，会弹出大图预览
4. **关闭预览**：点击预览窗口的关闭按钮或点击预览窗口外部区域

## 界面预览

- **主界面**：左侧为文件夹导航，右侧为表情包网格布局
- **文件夹导航**：显示所有表情包文件夹，按名称分类
- **表情包卡片**：显示表情包缩略图、文件名和大小
- **大图预览**：点击图片后显示大图，支持关闭操作

## 常见问题

### Q: 为什么无法显示表情包？
A: 请检查以下几点：
1. 确保 `.env` 文件中的 `EMOJI_DIR` 配置正确
2. 确保表情包文件夹存在且包含图片文件
3. 确保后端服务器已启动

### Q: 为什么图片显示不出来？
A: 请检查：
1. 图片格式是否为支持的格式（jpg, png, gif, webp, svg, bmp）
2. 图片文件是否损坏
3. 浏览器控制台是否有错误信息

### Q: 如何修改表情包文件夹？
A: 修改 `.env` 文件中的 `EMOJI_DIR` 配置，然后重启应用

### Q: 如何添加新的表情包？
A: 直接将表情包文件或文件夹添加到配置的表情包目录中，刷新页面即可看到新添加的表情包
