# 🌟 小朋友专属：表情包网站启动指南 🌟

嗨，小朋友！欢迎使用这个超可爱的表情包网站～ 这个指南会一步步教你怎么启动它，让你看到自己的表情包哦！😘

## 📋 准备工作

你需要准备：

1. 一台电脑（Windows系统）
2. 你的表情包文件夹（里面有很多可爱的图片）
3. 跟着我做的耐心和好奇心 ✨

## 🚀 第一步：安装魔法工具（Node.js）

Node.js是一个魔法工具，没有它，我们的网站就启动不了哦！

### 安装步骤：

1. 打开浏览器，输入这个网址：https://nodejs.org/zh-cn/
2. 点击那个绿色的「LTS」按钮（就是长期支持版本）
3. 下载完成后，找到下载的文件，双击打开
4. 按照提示，一直点击「下一步」，直到看到「完成」按钮
5. 点击「完成」，魔法工具就安装好啦！

## 📁 第二步：获取并找到你的项目文件夹

因为你是从GitHub下载的项目，所以需要先获取项目文件哦！

### 情况1：下载的是ZIP压缩包

1. 找到你从GitHub下载的ZIP文件（通常在「下载」文件夹里）
2. 右键点击这个ZIP文件，选择「解压到当前文件夹」
3. 解压完成后，你会看到一个新的文件夹，这就是我们的项目啦！

### 情况2：使用Git克隆的

1. 让爸爸妈妈帮你找一个合适的位置，比如「D盘」
2. 按照第四步的方法打开PowerShell窗口
3. 输入这个命令：`git clone https://github.com/你的用户名/你的仓库名.git`
4. 按「Enter」键，等待克隆完成

### 记住你的项目位置

找到解压或克隆后的项目文件夹，记住它的位置，后面会用到哦！

比如：`D:\表情包显示网站` 或者 `C:\Users\你的名字\Downloads\表情包显示网站`

## ⚙️ 第三步：配置你的表情包位置

1. 在项目文件夹里，找到一个叫 `.env` 的文件
2. 双击打开它（如果不知道用什么打开，就用「记事本」吧）
3. 你会看到一行字：`EMOJI_DIR = D:\A_收藏集\表情包`
4. 把等号后面的部分，改成你自己的表情包文件夹路径
   - 比如：你的表情包在 `D:\我的表情包` 里，就改成 `EMOJI_DIR = D:\我的表情包`
5. 保存文件（按Ctrl+S，或者点击文件→保存）

## 🎮 第四步：启动魔法命令

1. 按住键盘上的「Shift」键，同时在项目文件夹里右键点击空白处
2. 在弹出的菜单里，点击「在此处打开 PowerShell 窗口」（或者命令提示符）
3. 现在你会看到一个黑色的窗口，别害怕，这是我们的魔法控制台！

### 魔法步骤1：安装依赖

在黑色窗口里输入：

```
npm install
```

然后按「Enter」键

✨ 等待一会儿，你会看到很多文字在滚动，这是在安装魔法材料哦！

### 魔法步骤2：启动网站

安装完成后，输入：

```
npm run start
```

然后按「Enter」键

✨ 等待一会儿，你会看到两个网址：

- `http://localhost:3001`（这是后端魔法）
- `http://localhost:5173`（这是我们的网站！）

## 🌐 第五步：打开你的网站

1. 打开浏览器（比如Chrome、Edge）
2. 在地址栏输入：`http://localhost:5173`
3. 按「Enter」键
4. 哇！你的表情包网站就出现啦！🎉

## 🎯 怎么使用网站？

- **浏览表情包**：点击左边的文件夹名称，右边就会显示里面的表情包
- **搜索表情包**：在顶部的搜索框里输入你想找的表情包名字
- **看大图**：点击任意表情包，就会弹出大图
- **关闭大图**：点击右上角的「×」，或者点击空白处

## ❓ 遇到问题怎么办？

### 问题1：黑色窗口里出现红色的字

别担心！这可能是魔法咒语输错了，或者网络问题

- 检查你输入的命令是不是和我写的一样
- 检查网络连接
- 可以尝试重新输入命令

### 问题2：看不到表情包

- 检查.env文件里的路径是不是正确
- 确保你的表情包文件夹里有图片
- 确保图片格式是常见的（比如.jpg、.png、.gif）

### 问题3：网站打开是空白的

- 等待一会儿，可能正在加载
- 检查黑色窗口是不是还在运行（不要关掉它！）
- 刷新浏览器页面

## 🌈 小提示

- 黑色窗口要一直开着，网站才能运行哦！
- 想关闭网站时，在黑色窗口按「Ctrl+C」，然后输入「Y」按回车
- 记得经常保存你的表情包，网站会自动更新的！
